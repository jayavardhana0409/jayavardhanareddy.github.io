Nashville Housing Data Cleaning

This project entails the development of a comprehensive data cleaning script tailored for the Nashville Housing dataset. The script addresses various data preprocessing tasks, including the extraction and conversion of dates into a standardized format, handling null values by updating them based on specific criteria, and splitting address fields to improve data granularity. Additionally, it includes functionality to extract components from the OwnerAddress field, standardize values in the SoldASVacant column, and remove duplicate rows efficiently. With meticulous attention to detail and clear documentation, this script streamlines the data cleaning process, ensuring data integrity and consistency for further analysis. Users are advised to back up their data and customize the script according to their database structure and requirements before implementation.

Traffic Incidents

The Road Accident Dashboard is a comprehensive data analytics project designed to provide insights into road accidents occurring in 2021 and 2022. Through meticulous data cleaning and preprocessing, followed by in-depth analysis using Excel functions like pivot tables, the dashboard offers stakeholders a clear understanding of key performance indicators such as total casualties, accident severity distribution, and maximum casualties by vehicle type. With features including monthly trend analysis, casualties by road type and surface condition, and the relation between casualties by area/location and day/night, the dashboard facilitates informed decision-making in road safety planning and policy formulation. Its intuitive interface allows for dynamic exploration of the data, empowering stakeholders to identify trends, patterns, and areas for targeted intervention, ultimately contributing to the goal of reducing road accidents and minimizing casualties for safer communities.
